# 스크린샷 캡처 가이드

README가 참조하는 이미지 파일을 이 `docs/` 폴더에 아래 **파일명 그대로** 저장하면 자동으로 표시됩니다.
로그인이 필요한 화면은 데모 계정(`demo` / `demo1234`)으로 로그인 후 캡처하세요.

> Tomcat 실행 후 `http://localhost:8080/Petmunity` 기준 경로입니다.

| 파일명 | 화면 | 경로 |
|--------|------|------|
| `main.png` | 메인 | `/main/main` |
| `login.png` | 로그인 | `/user/login/form` |
| `register-person.png` | 회원가입 1단계(회원) | `/user/register_person/form` |
| `register-pet.png` | 회원가입 2단계(반려동물) | `/user/register_pet/form` |
| `adopt-list.png` | 입양·임보 목록 | `/community/adopt_community/adopt_community` |
| `petstar-list.png` | 펫스타그램 | `/community/petstar_community/petstar_community` |
| `info-list.png` | 정보 게시판 | `/community/info_community/info_community` |
| `message.png` | 쪽지함 | `/message/message` |
| `search.png` | 통합 검색 | `/search/search` |
| `mypage.png` | 마이페이지 | `/myPage/myPage` |

## 팁
- 게시판이 비어 보이면 데모 계정으로 글을 1~2개 작성한 뒤 캡처하면 보기 좋습니다.
- 캡처 후 파일명을 위 표와 똑같이 맞춰 `docs/`에 넣고 커밋하면 README에 바로 반영됩니다.
